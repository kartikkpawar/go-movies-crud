
## Running Project

To run this project

```
  go run main.go
```

### Buliding
To build this project

```
go build main.go
```

## Project Info
* Server will run on port 8000
* Routes on server
  * /movies -- GET -> Get all movies
  * /movie/id -- GET -> Get movie
  * /movie/id -- PUT -> Update movie
  * /movie/id -- DELETE -> Delete Movie
  * /movie/id -- POST -> Add Moive
